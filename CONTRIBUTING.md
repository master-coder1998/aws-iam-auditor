# Contributing

Thanks for your interest in improving this project. A few guidelines to get started:

- Fork the repository and create a feature branch: `git checkout -b feature/your-feature`
- Write tests for new features or bug fixes.
- Keep changes small and focused; add clear commit messages.
- Run `pytest` before submitting a PR.
- Open a Pull Request describing the change and tagging maintainers.

We appreciate constructive, well-scoped contributions.
